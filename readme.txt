Easily display bangla date (from bengali calendar) anywhere in your very own wordpress blog!


== Description == 
"Bangla Date Display" is a simple and easy to use wordpress plugin that allows you to show bangla date anywhere in your blog. Just install and activate the plugin. Then put a shortcode [bangla_date] in your blog post/page or sidebar where you want to show the current bangla date. Thats all!

== Installation ==

1. Upload the whole plugin folder to your /wp-content/plugins/ folder.
2. Go to the Plugins page and activate the plugin.
3. Put the shortcode [bangla_date] in your blog post/page or, sidebar.
